# BASICO

Visual Studio Code extension "BASICO". Support BASICO language.

## Features

- syntax highlighting

## Requirements

None.

## Extension Settings

This extension contributes the following settings:

* `basico.enable`: enable/disable this extension
* `basico.thing`: set to `blah` to do something
